# Giselle Sections Data

**Pure, type-safe data factories for section-based UIs.**

This package provides clean, reusable data factories that return fully typed props. No JSX. No presentation logic. Just pure data.

Designed as the single source of truth for all sections and components in Giselle UI projects — and usable in any React, Angular, Vue, or plain TypeScript application.

## Status
- **In active development**
- Currently being extracted and refined from my personal portfolio project
- First public release planned after the main portfolio site is live (expected May/June 2026)

## Features
- Fully typed data factories with TypeScript 5.9 (strict mode)
- Zero runtime dependencies
- Framework agnostic (works with any frontend framework)
- Easy to test in isolation
- Designed to be migrated to a real database or API layer later without changing consumers

## Installation

```bash
npm install @alexrebula/giselle-sections-data
```
